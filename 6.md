# weui-page
  ## btn页面
  ### 先写结构
  -优秀的套路
  .page__hd+.page__bd
  ### BEM 命名规范 国际规范
  -.page 页面
   开启一个区块block
  -分上下两部分
   .page__hd头部
   .page__bd主体
   hd、bd是block下的element
   Block和Element之间用__分隔
   -Modifier 修饰符
    不同的类用不同的状态
     -.weui-btn-primary 主要按钮
     -.weui-btn-default 普通按钮
     -.weui-btn-link 链接按钮
   使用语义化标签对搜素引擎更好
  ### BEM的优势
   -国际规范大家都遵守
   -结构清晰，搭建盒模型方便
    -有一个区块要表达block
    -有几个元素要表达element
  -简单易读
  -解决了写页面难命名的问题
   最简单的英文单词，结构相关
   .page
    .page__hd
    .page__bd
   没有BEM命名规范，要写很多单词 
  ### 再写样式
   #### css规则
    -css重置样式
     有些元素默认样式不同，需要重置
     把页面变成一张干净的白纸
     通配符选择器，选择所有元素，性能不好
     * 选择所有元素
     * {
      margin: 0;
      padding: 0;
     }
