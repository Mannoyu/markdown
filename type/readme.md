# JS 有多少种数据类型？
 八种数据类型
 根据ECMA262 规范，primitive 原始数据类型有6种，分boolean string null undefined symbol numeric，
 其中 numeric 再可分为number 和es6 新增的 bigint，还有一种 object 引用类型

## 简单数据类型
 原始数据类型(primitive types)
 - Number 数值
 - String 字符串
 - Boolean 布尔值
 - null
   - 用于表示一个有意设置为空的对象引用(空的引用)
   - 表示某处应该有一个值，不过目前没有(空的值)
   - 可以用于清空一个变量的值，释放内存，垃圾回收
 - undefined
   - 未定义，表示变量未被初始化或不存在
   - 当申明一个变量但未赋值时，默认值是 undefined
   - 某个对象的属性不存在时
   - 函数没有返回值时
   - 访问不存在的数组索引时
 es6 以前 js 只有6种类型
### es6 新增的类型
 - symbol 唯一
 - bigint 大数

## 复杂数据类型
 - object
 引用数据类型 (内存分配)

## 内存分配去思考
 - 冯洛伊曼 现在计算设备
   运算器、控制器、存储器、输入设备、输出设备
 - 代码 外存 硬盘
 - 编译，代码从硬盘调入内存中
 - 执行上下文 变量环境，词法环境，执行代码
 - 推入**调用栈** 在栈内存中分配空间
   快，空间小
   变量环境/词法环境 原始数据类型 直接写入值
   编译阶段给他的空间正好合适  函数执行上下文对象要占用多少空间是能算出来的，
   当一个函数执行完，出栈，很快的算出新的栈顶元素位置，指针偏移量就可以切换上下文的执行
   快速、稳定、扩展
