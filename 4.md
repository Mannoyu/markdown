## var let const
早期的js 变量声明，var 会提升到作用域顶部，可以跨块级作用域使用，没有常量，用代码规范约束
var PI = 3.14;
var 不支持块级作用域
js设计的时候比较赶工 浏览器的副产品
js没有经过深思熟虑的，有一些瑕疵

let 负责变量，const常量
支持块级作用域
变量的类型由值类型决定
let const 不会提升到作用域顶部，会报错


## for + setTimeout
