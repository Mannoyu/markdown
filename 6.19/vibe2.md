📼 「磁带墙」
每天录一盒磁带。一年后你有一整面磁带架。抽出任何一盒，那天都在里面。

🏗️ 核心概念
隐喻	含义
磁带架	你的时间——整面墙的磁带按行排列
每一盒磁带	一天
磁带脊颜色	那天的情绪主色
磁带标签	日期 + 一句话摘要
磁带内容	照片 / 声音 / 一句话 / emoji
空白磁带	没有记录的日子，透明壳微微发灰
每周精选	AI 从 7 盒中挑一盒"本周最佳"放在顶部
🎯 每日使用（30 秒）
打开 app → 磁带架全景，今天的空位微微发光
选情绪颜色 → 磁带脊着色
可选：一句话 / 照片 / emoji / 录 5 秒声音
手指一推 → 磁带滑入架子，"咔"一声嵌入
完成
✨ 惊艳瞬间
时刻	视觉
开屏	磁带架从暗处亮起，每盒磁带脊折射出微弱光泽
今日磁带发光等你	空位有呼吸光效，引导你填
推入磁带	磁带滑入架位 + "咔嗒"震动反馈 + 涟漪扩散
抽出旧磁带	点一盒 → 磁带从架子弹出一小段 → 画面展开那天的记忆卡片
周总结	7 盒磁带自动排成一行 → AI 一句话总结浮现
切换月份	架子横向滑动，月份标签切换
🎨 视觉风格
复古未来感：磁带是真实磁带的比例微缩 + 现代扁平 UI 的结合
配色：暗灰底 + 磁带本身米白/透明壳 + 情绪色做脊标
光效：柔光、粒子、磁带塑料壳的反光质感
